## Goal

## Result
- Added a small prompt store that loads provider wrappers from files and prepends them to the base prompt templates.

- Documented the current prompt layout in `docs/PROMPT_ARCHITECTURE.md`.

## Known Issues
- The repository still has many older prompt strings outside the longform flow.
- Only the longform script-plan, script-generation, and image-prompt prompts were moved into the new wrapper structure.
- The current wrappers are intentionally small; future tasks may need richer Claude tuning once live validation shows a need.
