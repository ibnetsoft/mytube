웹툰 자동 Crop 및 분할 원칙 (AI 엔진)

1 사다리꼴 및 비정형 처리

기하학적 왜곡이 있는 컷이나 사다리꼴 형태의 연출도 AI가 외곽선을 추적하여, 그림의 가장 바깥쪽 경계를 기준으로 단 1px의 잘림 없이 온전하게 추출합니다.

2 화이트/블랙 여백 제로화

컷 주변의 불필요한 흰색/검은색 여백을 0.1px 단위로 정밀하게 도려냅니다. 그림 내용만 꽉 차게 추출하여 영상화 시 몰입감을 극대화합니다.

3 중복 이미지 자동 제거

동일한 장면이 중복 추출되는 현상을 방지하기 위해 시각적 해시(Vision Hash) 비교를 거쳐 내용이 같은 컷은 자동으로 하나만 남기고 제거합니다.

4 가로형 컷 전체 보존 (L-Box)

가로로 긴 영화적 연출 컷은 크롭하지 않고 온전하게 추출하되, 숏폼 비율(9:16) 규격에 맞춰 레터박스를 적용하여 전체 내용을 보존합니다.

5 개별 컷 독립 분할 원칙

장면 병합 로직을 배제하고, 웹툰의 모든 의미 있는 컷은 독립된 하나의 장면으로 분할하여 정확한 TTS 매칭과 모션 제어를 보장합니다.

6 측면(Side) 여백 완벽 제거

상하뿐만 아니라 좌우측에 남은 편중된 여백도 시각적으로 분석하여 제거합니다. 일러스트가 한쪽으로 쏠리지 않도록 비정상적인 측면 공간을 스캔하여 배제합니다.

📐 Webtoon Production & Crop Master Guide

Type 1: Vertical Long (세로형)
  • • Effect: pan_down, pan_up
  • • 너비를 화면에 맞추고(Fit Width) 높이 방향으로 위아래 전체를 훑는 다이나믹한 연출
  • • 배경 공개나 전신샷에 최적화
Type 2: Horizontal Wide (가로형)
  • • Effect: pan_left, pan_right
  • • 높이를 화면에 맞추고(Fit Height) 좌우로 이동하며 파노라마처럼 전체 배경을 노출
  • • 시네마틱한 풍경이나 긴 배경 컷에 사용
Type 3: Standard (일반)
  • • Effect: zoom_in, zoom_out
  • • 화면 가득 채운 후 중심(또는 얼굴)을 향해 서서히 확대/축소
  • • 일반적인 대화나 클로즈업 컷에 범용적으로 사용
Type 4: Static (고정)
  • • Effect: static
  • • 고정된 화면 유지. 대사가 길거나 짧은 전환 컷에 적합
  • • 2D 이미지가 그대로 노출되는 가장 깔끔한 연출
💡
Pro Tip: 기획서 자동 생성을 이용하면 이미지의 가로세로 비율을 AI가 스스로 판단하여 위의 효과들을 자동으로 배정합니다. 가로가 매우 넓은 컷은 자동으로 TYPE 2 좌우 패닝이 적용되어 원본을 거의 손실 없이 보여줍니다.
